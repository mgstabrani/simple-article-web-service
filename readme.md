# simple-article-web-service
> A simple article web service using nodejs and mongodb

## Technologies
- NodeJS
- NPM
- Mongodb
- HapiJS

## Installation
- Install dependency
```
npm install
```

## Running Local (dev mode)
### Environment Variables
```
SERVER_HOST=localhost
PORT=3000
DB_HOST=localhost
DB_PORT=27017
DB_NAME=articles
```
### Commands
#### Running service in development mode
```
npm run dev
```
#### Integration test
```
npm run test
```
